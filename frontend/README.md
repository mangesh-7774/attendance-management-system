# 🎓 Student Attendance Management System

A full-stack MERN application designed to manage student attendance efficiently with role-based access for Admin, Teacher, and Student.


## 🚀 Live Demo

🔗 Frontend: https://attendance-management-system-tau-five.vercel.app/
🔗 Backend API: https://attendance-management-system-5dlc.onrender.com


## 🛠️ Tech Stack

* Frontend: React.js, Tailwind CSS
* Backend: Node.js, Express.js
* Database: MongoDB Atlas
* Charts: Recharts
* Authentication: JWT

---

## 🔑 Features

### 👨‍💼 Admin

* Register teachers (auto Employee ID generation)
* Manage teachers (edit/delete)
* Auto class creation and teacher assignment
* View attendance analytics (charts)
* Generate reports (PDF & Excel)
* Manage academic calendar (holidays & non-working days)

### 👩‍🏫 Teacher

* First-time login via Email OTP verification
* Add / edit / delete students
* Mark attendance
* View attendance analytics
* Generate reports
* Identify detained students with custom criteria (75%, 70%, 65%)

### 🎓 Student

* View attendance dashboard
* Monthly attendance reports
